启动一个 Node 服务器

如果没有 Docker，如何启动一个 Node 器呢？

这件事情很简单，只需要把下面这段代码保存为一个独立的文件，如 app.js，然后在这个文件的目录运行 node app.js 就可以了。

const http = require('http');

const hostname = '127.0.0.1';
const port = 3000;

const server = http.createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World');
});

server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});

代码来自于这个网址

使用 Docker 也可以完成这件事，不过我们需要修改一些地方。我们需要把 hostname 的值修改为 0.0.0.0。

然后运行如下命令：

docker run -it -v "$PWD":/usr/src/app  node:14 node app.js

之后，你通过浏览器访问 localhost:3001 就同样可以访问的启动的服务器。

我们来解析一下这个命令，每次使用 docker run 命令都会启动一个 Docker 容器，而一个容器，就可以被看做一个独立的主机。-v 做的是本地与此容器之间的文件映射，node:14 自然就是指定我们将使用 node 这个镜像，而 node 的版本是 14。在此之后的命令便是启动好容器之后要运行的命令。

除了 docker run 命令外，我们还可以通过另外一种 docker build 运行一个容器。
