概述

Docker 对很多前端程序员比较陌生，究其原因，是我们可以使用到他的场景实在是太少了。如果你经历过没有 Docker 部署服务器的年代，你肯定大呼，Docker 拯救了你的生命。

关于 Docker 每个命令如何使用，我相信在你下载 Docker 之后，学会以后 docker [OPTIONS] --help 来查手册，就基本可以全部了解了。但最难的是哪一部分呢？是不知道 Docker 的使用场景是什么，以至于在可以使用 Docker 的时候往往选择了其他麻烦的方法。

今天，我们就来分享三个场景，通过这三个场景，我们就能大致了解 Docker 能做什么，它的边界在哪，在需要用到的时候就可以随时拿出来用了。
